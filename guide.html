"Politique de Confidentialité pour le Bot Discord"

Introduction La présente politique de confidentialité décrit comment ZyriaIA collecte, utilise, et protège les informations des utilisateurs sur Discord. En utilisant ZyriaIA, vous acceptez les pratiques décrites dans cette politique. Informations Collectées ZyriaIA peut collecter les types d'informations suivants : Données d'utilisateur : Identifiants de compte Discord (nom d'utilisateur, identifiant) Messages envoyés au bot Données d'interaction : Commandes utilisées Fréquence d'utilisation Temps d'interaction Utilisation des Informations Les informations collectées sont utilisées pour : Améliorer l'expérience utilisateur Fournir des réponses précises aux requêtes Analyser l'utilisation du bot pour des mises à jour futures Protection des Informations Nous prenons des mesures de sécurité appropriées pour protéger vos informations : Cryptage des données : Les informations sensibles sont cryptées. Accès limité : Seules les personnes autorisées peuvent accéder aux données. Mises à jour régulières : Le bot est régulièrement mis à jour pour corriger les vulnérabilités. Partage des Informations ZyriaIA ne partagera pas vos informations avec des tiers sans votre consentement, sauf si cela est requis par la loi ou pour protéger les droits de l'entreprise. Droits des Utilisateurs Vous avez les droits suivants concernant vos informations : Accès : Vous pouvez demander à voir les informations que nous avons sur vous. Modification : Vous pouvez demander à modifier des informations inexactes. Suppression : Vous pouvez demander la suppression de vos données. Modifications de cette Politique Cette politique de confidentialité peut être mise à jour de temps à autre. Nous informerons les utilisateurs des modifications significatives via des annonces sur Discord. Contact Pour toute question concernant cette politique de confidentialité, veuillez contacter l'équipe de ZyriaIA via le canal approprié sur Discord. Merci d'utiliser ZyriaIA ! Nous nous engageons à protéger votre confidentialité.
